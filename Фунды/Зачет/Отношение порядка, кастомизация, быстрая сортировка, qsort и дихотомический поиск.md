# 1. Отношение порядка на пространстве элементов
Отношение порядка — это бинарное отношение $\leq$  на множестве, которое обладает тремя свойствами.
#### Свойства
1. **Рефлексивность**: $\forall a: a \leq a$ 
2. **Антисиммитричность**: $\text{if } a \leq b \text{ and } b \leq a \Rightarrow a = b$
3. **Транзитивность**: $a \leq b \ \ b \leq c \ \ a = c$

**Важная деталь**: В большинстве алгоритмов сортировки используется строгое отношение порядка (<), которое обладает:
- **Иррефлексивностью**: ∀a: ¬(a < a)
- **Транзитивностью**
- **Асимметричностью**: если a < b, то ¬(b < a)
